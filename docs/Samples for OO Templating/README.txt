English:

You'll find here 

1) a sample template for project related exports to an OpenOffice template. 
With it you can produce a report like this

A header line with the report time
	For each project a block with
		For each project member (with booked times) a block with 
			For each day a block with
				Tasks ordered by time
			Day ends with a day sum
		Project member block ends with a user sum
	Project ends with a project sum
All closed by a grand total

You can use this template as a base for your company specific export templates
and upload it during each export. Or in admin mode upload it permanently for 
regular use.


2) a similar template with a small modification to skip all tasks with 
"calculate time"=NO (for instance : 'Leave') 

3) A brief description of the data available for your own templates


German:

Sie finden hier 

1) ein beispielhaftes Template für projektbezogene Exports über
ein Open Office-Template. Sie könne damit eine Auswertung analof folgender
Beschreibung erzeugen:

Eine Kopfzeile mit dem Leistungszeitraum
	Für jedes Projekt einen Block mit
		Für jeden Mitarbeiter einen Block mit
			Jeweils einem Block je Tag mit
				Den erfassten Tätigkeiten in chronologischer Reihenfolge
			Der Tag schließt mit einer Tagessumme
		Der Mitarbeiter schließt mit einer Mitarbeitersumme
	Das Projekt schließt mit einer Projektsumme
Ganz am Ende kommt eine Gesamtsumme für den Leistungszeitraum.

Sie könne dieses Template als Basis für Ihr Unternehmenstemplate (mit eigenem Logo
eigener Formatierung etc.) benutzen. Dieses Tenplate könne sie dann bei jedem Export-
Vorgang hochladen, bzw. auch permanent zur regelmässigen Verwendung zur Verfügung 
stellen, wenn sie einmalig im Admin-Mode den Upload-Vorgang während eines Exports
durchgeführt haben.

2) ein ganz ähnliches Template, das alle Tätigkeiten mit "Zeit berechnen"=NEIN
ausfiltert (z.B. "Leave"/"Gehen")

3) eine Kurzbeschreibung des Datenstroms, den Sie in eigenen Templates verwenden können.
